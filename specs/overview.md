# Todo App Overview

## Purpose
Transform console todo into multi-user web app with persistent storage.

## Current Phase
Phase II: Full-Stack Web Application.

## Tech Stack
- Frontend: Next.js 16 (App Router), Tailwind 4
- Backend: FastAPI, SQLModel
- Database: Neon Serverless PostgreSQL
- Auth: JWT (to be aligned with Better Auth token issuance)

## Features
- Task CRUD with completion toggle
- User authentication
- User isolation per JWT subject
